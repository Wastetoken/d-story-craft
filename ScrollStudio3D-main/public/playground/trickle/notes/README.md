# ScrollStudio - Immersive 3D Scroll Experience

## Overview
An immersive web experience featuring a 3D GLB model with scroll-triggered animations, cinematic post-processing effects, and elegant typography.

## Features
- 3D GLB model with embedded animations
- GSAP ScrollTrigger for scroll-based movement
- UnrealBloom post-processing for cinematic glow
- Custom cursor with GSAP tracking
- Noise overlay texture
- Responsive design

## Tech Stack
- Three.js r128 for 3D rendering
- GSAP 3.12 with ScrollTrigger
- React 18
- Tailwind CSS

## Files Structure
- `index.html` - Main entry with Emotive Creature experience
- `about.html` - About page with Skateboarder experience
- `styles.css` - Custom styles and CSS variables
- `app.js` - Main React application
- `about-app.js` - About page React application
- `components/` - Reusable UI components
- `components/about/` - About page specific components
